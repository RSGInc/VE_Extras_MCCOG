# MCCOG_VE_Extras
MCCOG Version of VE-Extras, designed to install and run models and packages specific to MCCOG and the Central Indiana VisionEval model. This includes customized powertrains packages (base and multiscenario models), Population Sim, and the driverless, multimodal, and work from home packages for VisionEval 3.0. The VEIndyRSPM package contains model instances for MCCOG. This has not yet been tested for compliance with VE 4.0 which is being developed as of June 2025.

NOTE:
WHEN INSTALLING MODELS FROM VEIndyRSPM PACKAGE, USER MUST UNZIP THE OUTPUT_2019 AND OUTPUT_2050 FOLDERS IN "inputs/VEPopulationSimInputs" BEFORE RUNNING THE MODEL.
